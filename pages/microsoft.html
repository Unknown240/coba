Cara memverifikasi keaslian file?

  • Kita dapat menggunakan metode hash file untuk memverifikasi apakah sebuah file asli atau tidak, tetapi kita harus tahu checksum sebenarnya dari file tersebut.

  • Microsoft tidak menerbitkan daftar checksum pada Link publik untuk file-file mereka, namun, ini tersedia untuk pelanggan berlangganan berbayar dari MVS.

  • Ada banyak tempat di mana kita bisa menemukan checksum untuk file-file Microsoft dan membandingkannya dengan file-file kita menggunakan alat-alat seperti 7-Zip (Setelah menginstal 7-Zip, klik kanan pada file ISO dan pilih 7-Zip > CRC SHA). Berikut adalah daftar situs yang berisi checksum.

    1. files.rg-adguard (koleksi paling lengkap)
    2. MVS dump
    3. genuine-iso-verifier
    4. msdn.rg-adguard
    5. sha1.rg-adguard
